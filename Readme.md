# 🧠 Mind Quiz App

A modern interactive quiz application built using HTML, CSS, and JavaScript.

## 🚀 Features

* 🎯 Multiple category questions
* ⏱️ Timer with circular animation
* 📊 Progress tracking
* 🔊 Sound effects
* 🎉 Confetti celebration
* 🔥 Streak-based scoring system
* 💻 Clean and modern UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)
